# Trading DashBoard v7.0

## 🚀 개요
Trading DashBoard는 개인 트레이더를 위한 **올인원 실전 트레이딩 운영 시스템**입니다. 단순한 기록을 넘어, **매매 전 계획(Plan) → 보유 중 대응(Execute) → 종료 후 복기(Review)** 과정을 하나의 유기적인 루틴으로 연결하여 트레이더의 실행 품질과 심리적 안정을 돕습니다.

v7.0에서는 더욱 정교해진 통계 분석과 프리미엄 UI 디자인, 그리고 직관적인 차트 흐름 분석 기능이 강화되었습니다.

## 🎯 핵심 목적
1. **계획 중심 매매**: 진입 전 논리와 손절/목표가를 명확히 설정하여 뇌동매매를 방지합니다.
2. **리스크 철저 통제**: 가동 자산 대비 허용 리스크와 레버리지를 수치로 확인하고 체계적으로 관리합니다.
3. **대응의 기록**: 홀딩 중 발생하는 심리 변화와 주요 대응(본절 이동, 부분청산 등)을 타임스탬프와 함께 남깁니다.
4. **패턴의 자산화**: 종료된 매매를 셋업과 실수 태그로 분류하여, 나만의 필승 전략(Playbook)을 구축합니다.

---

## 🛠 주요 기능 및 흐름

### 1️⃣ Overview: 시장 상황 및 성과 요약
- **Premium Metrics**: 핵심성과지표(Net PnL, 승률, Avg R 등)를 한눈에 파악합니다.
- **Dynamic Curves**: 개선된 Equity/Balance Curve를 통해 자산 흐름을 시각적으로 모니터링합니다.
- **Heatmap Calendar**: 월간 성과를 시각적으로 확인하고 특정 일자 매매로 즉각 이동합니다.
- **Quick Launch**: 자주 사용하는 외부 도구(거래소, 차트 등)를 중앙에서 관리합니다.

### 2️⃣ Journal: 단계별 트레이딩 워크플로우
- **Pre-Trade Context**: 진입 논리, 무효화 기준, 체크리스트를 점검하는 '매매 전 선언' 섹션입니다.
- **Risk & Execution**: 현재가, 손절가, 리스크(%)를 기반으로 주문 규모와 분할 진입 전략을 설계합니다.
- **Live Management**: 포지션 보유 중 발생하는 시그널과 심리 상태를 실시간으로 기록합니다.
- **Post-Trade Assessment**: 최종 PnL, R-Multiple, 수수료, 계좌 영향을 자동 계산하고 복기를 완료합니다.

### 3️⃣ Library & Playbook: 데이터 자산화
- **Trade Library**: 방대한 기록 중 필요한 매매를 셋업, 종목, 실수별로 정밀 검색합니다.
- **A+ Playbook**: 우수한 성과를 낸 매매를 갤러리 형태로 보관하여 최상의 매매 감각을 유지합니다.

---

## 💡 실전 사용 팁
- **계획 우선**: 진입 후 기록하는 것이 아니라, **진입 전에 계획을 먼저 작성**하는 것이 가장 효과적입니다.
- **레버리지의 이해**: 레버리지는 수익을 키우는 마법이 아니라, **필요 증거금과 체감 변동성을 관리하는 변수**로 사용하세요.
- **JSON 백업**: 모든 데이터는 브라우저(로컬)에 저장되므로, 주기적인 JSON 내보내기를 권장합니다.
- **차트 증거**: 타임프레임별 차트 링크를 충실히 남길수록 복기 시 더 깊은 인사이트를 얻을 수 있습니다.

---

## ⚠️ 주의사항
- 본 도구는 **데이터 자동 수집 기능을 포함하지 않습니다**. 모든 가격 및 수치는 트레이더가 직접 입력하고 관리해야 합니다.
- 데이터는 LocalStorage와 IndexedDB를 병용하여 영속성을 유지하지만, 브라우저 캐시 삭제 시 데이터가 소실될 수 있으므로 정기 백업이 필수입니다.
