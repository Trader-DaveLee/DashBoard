Journal Form

나만의 매매 원칙을 항상 띄워두고, 뇌동 매매를 방지하는 체크리스트로 활용하세요.

Library

수익이 큰 거래보다 원칙을 잘 지킨 거래(Grade S/A)를 모아 나만의 필승 패턴을 정립하세요.

추천 루틴

매매 전
  • 체크리스트 통과 여부 확인
  • 손절이 명확한지 확인
  • 리스크(%)와 레버리지 확인
  • 목표가가 현실적인지 확인
매매 후
  • 종료 시간 입력
  • 실수 태그 최소 1개라도 점검
  • Review에 다음 매매 교정 포인트 한 줄 작성

주의할 점

  • 이 도구는 실행 보조와 복기용입니다. 시장 데이터를 자동으로 받아오지 않기 때문에 가격/손절/목표 입력은 사용자가 책임지고 넣어야 합니다.
  • 로컬/브라우저 저장 구조이므로 정기 백업(JSON 내보내기)를 습관화하는 것이 좋습니다.
  • 차트는 링크 중심으로 기록하세요.
